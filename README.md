## Getting Started

### Dependencies

-   Python 3.8+
-   pipenv
-   pytest
-   pytest-cov
-   pylint

### Installing

-   Clone the repository

### Setup Wheel

```bash
python setup.py bdist_wheel
```

### Lint this project

```bash
pylint --rcfile=.pylintrc MelodieFuncFlow
```
